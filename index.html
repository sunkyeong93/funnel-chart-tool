채널 퍼널 차트 — 매트릭스 / PC vs Mobile

한 데이터로 두 가지 차트를 토글해서 추출 — (1) 채널 포지션 매트릭스 (2) PC vs Mobile 가로 막대. SVG로 다운로드해 Figma에 드래그하세요.

데이터 갱신
또는 ↓ 표 붙여넣기

엑셀(.xlsx)을 올리면 자동 파싱(템플릿 / GA4 피벗 / 전치 피벗 모두 지원). PC와 MO를 자동 분리해서 두 차트가 같은 데이터로 그려집니다.

Figma 사용법: 다운로드한 SVG를 Figma 캔버스에 드래그하면 모든 요소(버블·막대·라벨·축)가 벡터 레이어로 들어옵니다.

퍼널 단계별 이탈률 분석

단계별 통과율 영역 차트 + 단계 사이 Drop off 박스. 종합/PC vs Mobile 두 가지 차트 토글. SVG 다운로드해 Figma에 드래그하세요.

데이터 갱신
또는 ↓ 표 붙여넣기

템플릿 / GA4 피벗 / 전치 피벗 형식의 엑셀(.xlsx) 모두 자동 인식. 5단계 + PC·MO 디바이스로 분리해서 영역 차트와 Drop off 박스를 그립니다.

Figma 사용법: 다운로드한 SVG를 Figma 캔버스에 드래그하면 단계 화살표·곡선·라벨·Drop off 박스가 모두 벡터 레이어로 들어옵니다.

유형별 유입 · 가입 · 최종 전환율

유형별 유입수와 최종 전환수를 막대로, 전환율(%)을 라인으로 함께 보여주는 콤보 차트. SVG로 다운로드해 Figma에 드래그하세요.

데이터 갱신
또는 ↓ 표 붙여넣기

유형 / 단계별 / 카운트 형식의 엑셀(또는 TSV)을 올리세요. 각 유형마다 "유입" 행과 "가입신청완료/주문서접수완료/구매" 등 최종 전환 단계 행이 페어로 들어 있어야 합니다.

Figma 사용법: 다운로드한 SVG를 Figma 캔버스에 드래그하면 모든 막대·라인·라벨이 벡터 레이어로 들어옵니다.

랜딩별 비교 — 유형별 / 디바이스별

같은 유형이지만 랜딩이 다른 두 캠페인의 전환율 비교. 너겟 전체값은 회색 가이드로 표시. SVG로 다운로드해 Figma에 드래그하세요.

데이터 갱신

"디바이스별 전체 너겟 퍼널 상세 분석" + "너겟 유형별 퍼널 상세 분석" 형식의 엑셀을 인식합니다. 전체/MOWEB/PC 컬럼 + 1) 너겟 메인, 2) 기획전 (...) 두 서브섹션 구조.

Figma 사용법: SVG를 드래그하면 모든 영역·라인·라벨이 벡터 레이어로 들어옵니다.
