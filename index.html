두 개의 GA4 퍼널 데이터를 비교 차트 형식으로 변환해줘. [사전 준비] 아래 두 데이터를 각각 붙여넣고, - 비교할 캠페인/유형 이름을 명시해줘 (예: "단말별 전체", "모바일상시") - 기준이 되는 전체 데이터도 있으면 함께 붙여넣어줘 [처리 규칙] 1. 각 데이터에서 퍼널 단계를 순서대로 찾아줘 - 숫자 접두사(1., 2. 등)나 기호(①②)가 있으면 순서 기준으로 사용 - 단계명은 로우 데이터에 있는 그대로 사용 (절대 임의로 바꾸지 말 것) 2. 두 데이터의 단계 수가 다를 경우 → 더 많은 쪽 기준으로, 없는 단계는 0으로 채워줘 3. 전체/MO/PC가 구분된 경우 각각 추출 (없으면 전체만) 4. 수치는 정수만 (소수, %, 콤마 제거 / 비율값은 제외하고 사용자 수만 사용) 5. 날짜별 열이 여러 개면 → 총합계 열만 사용 [출력 형식 — 탭 구분, 그대로 복사 가능하게] [1] 전체 기준 (기준 데이터명 또는 "전체합산") 행 레이블 전체합계 MO합계 PC합계 {1단계명} {전체} {MO} {PC} {2단계명} {전체} {MO} {PC} {3단계명} {전체} {MO} {PC} {4단계명} {전체} {MO} {PC} {5단계명} {전체} {MO} {PC} 1) {캠페인명1} {1단계명} {전체} {MO} {PC} {2단계명} {전체} {MO} {PC} {3단계명} {전체} {MO} {PC} {4단계명} {전체} {MO} {PC} {5단계명} {전체} {MO} {PC} 2) {캠페인명2} {1단계명} {전체} {MO} {PC} {2단계명} {전체} {MO} {PC} {3단계명} {전체} {MO} {PC} {4단계명} {전체} {MO} {PC} {5단계명} {전체} {MO} {PC} [데이터 1 — 기준 또는 캠페인1] (여기에 첫 번째 파일 데이터 붙여넣기) [데이터 2 — 캠페인2] (여기에 두 번째 파일 데이터 붙여넣기) [처리 순서] Step 1. 두 데이터에서 퍼널 단계와 수치를 파악한 후, 아래 내용을 먼저 나에게 보여줘: - 캠페인1 이름 / 단계명 목록 / 각 단계 전체·MO·PC 수치 - 캠페인2 이름 / 단계명 목록 / 각 단계 전체·MO·PC 수치 - 두 캠페인의 단계 매핑 방식 (단계명이 다른 경우 어떻게 대응시켰는지) Step 2.
